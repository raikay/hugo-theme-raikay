# 方法一

【项目】  --》 【加载库】  --》【管理库】

![](http://blogimg.raikay.com/330638699345874944.png)

**安装**

弹出窗口后，搜索、安装即可

![](http://blogimg.raikay.com/330638636074799104.png)

**查看示例代码**

安装之后， 【文件】 --》【示例】 ，就可以查看库带的示例代码，大多数库都有，这个示例代码是很好的教程。

![](http://blogimg.raikay.com/330638610615373824.png)



# 方法二
